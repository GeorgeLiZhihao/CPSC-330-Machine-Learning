# CPSC 330 – Applied Machine Learning

> This repository is part of my machine learning and data science portfolio, alongside projects in optimization and statistical modeling.

This repository contains a collection of assignments completed for **CPSC 330: Applied Machine Learning** at the **University of British Columbia (UBC)**.

The work focuses on developing practical machine learning skills through hands-on implementation of data preprocessing, model building, evaluation, and interpretation using real-world datasets.

---

## 🚀 Key Topics Covered

* Data preprocessing and feature engineering
* Supervised learning (regression and classification)
* Model evaluation and validation techniques
* Hyperparameter tuning and model selection
* Ensemble learning methods
* Model interpretation and explainability

---

## 🛠️ Tools & Technologies

* Python (NumPy, Pandas, scikit-learn)
* Jupyter Notebook
* Matplotlib / Seaborn
* SHAP

---

## 🎯 Learning Outcomes

Through this work, I developed:

* A solid understanding of core machine learning concepts
* Practical experience with end-to-end ML workflows
* The ability to evaluate and compare models effectively
* Experience interpreting model outputs and communicating results
